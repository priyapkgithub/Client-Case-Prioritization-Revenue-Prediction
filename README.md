 # Legal Case Prioritization & Revenue Prediction
Project Highlights:
Synthetic Dataset Generation
Created a realistic dataset of 1,000+ legal case records using Python (NumPy, Pandas), simulating real-world parameters like case complexity, urgency, and internal resource availability.

ML Pipeline Development
Built a Random Forest Classifier to predict case priority (High/Medium/Low). Achieved ~85% accuracy, with clear insights from the confusion matrix and feature importance.

Business Integration with Power BI
Exported model outputs to a Power BI dashboard to visualize:

Case status and predicted priority

Lawyer workload balancing

Potential revenue segmentation
This enabled data-driven decision-making for client onboarding and staff allocation.

Tools Used: Python, scikit-learn, Pandas, Matplotlib, Seaborn, Power BI
